**Investor Quiz App**
- A Flutter mobile application developed as part of an OJT/Internship project.

**Features**
1. Lessons & Quizzes – Access educational lessons and take quizzes to test your knowledge.
2. Score Monitoring – Keep track of quiz scores and progress over time.
3. Settings – Customize app preferences for a personalized experience.
4. Intuitive UI/UX – Designed with Figma for a smooth user experience.
5. Developed with Dart and Flutter
6. Available for Android devices

**Technologies Used**
- Flutter – Cross-platform mobile development framework
- Dart – Programming language for Flutter
- Figma – UI/UX design
